# Apa yang Dipelajari di Day 1

Di hari pertama ini, kita fokus pada **dasar-dasar JavaScript** dan cara menghubungkannya dengan HTML untuk membuat interaksi sederhana. Project ini membiasakan kita dengan konsep fundamental sebelum masuk ke fitur yang lebih kompleks seperti array atau tipe data lainnya

## Konsep Utama yang Dipelajari

- **Variabel**: Mendeklarasikan variabel menggunakan `const` untuk menyimpan referensi ke elemen HTML (misalnya, `const nameInput = document.getElementById('nameInput')`).
- **Fungsi**: Membuat fungsi sederhana (`generateGreeting()`) yang mengambil input, memprosesnya, dan menghasilkan output.
- **Manipulasi DOM**: Menggunakan `document.getElementById()` untuk mengakses elemen HTML, dan `textContent` untuk mengubah isi elemen secara dinamis (misalnya, mengubah teks di div `#output`).
- **Event Listener**: Menambahkan event handler dengan `addEventListener('click', ...)` untuk merespons aksi pengguna (klik tombol).
- **Kondisi Dasar**: Menggunakan `if-else` untuk validasi sederhana (cek apakah input kosong).
- **Template Literals**: Menggunakan backticks (`` ` ``) untuk string interpolation (misalnya, `` `Halo, ${name}!` ``).
